# Tutorials

Use this section for learning-oriented lessons.

A tutorial should:

- Teach by doing, end to end.
- Assume minimal prior knowledge.
- Build confidence with small, sequenced steps.
- Focus on one learning goal per page.

Typical examples:

- "Your first Grimoire spell"
- "From compile to simulate"
- "Adding your first venue adapter"
